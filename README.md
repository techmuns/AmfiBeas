# AmfiBeas

Live operating and financial dashboard for Indian Asset Management Companies (AMCs).

## Stack

- Next.js 15 (App Router) + TypeScript
- Tailwind CSS
- Recharts for visualizations
- lucide-react icons

## Scripts

```bash
npm install
npm run dev        # http://localhost:3000
npm run build
npm run lint
npm run typecheck
```

## Structure

- `src/app` — routes (App Router)
- `src/components` — UI + layout components
- `src/lib` — utilities (formatters, classnames)
- `src/data` — types and demo fixtures

## Status

Phase 1 scaffold. Charts, fixtures, and routes land in subsequent phases.
